# **App Name**: Oneness Kingdom Landing

## Core Features:

- Landing Page: A visually appealing landing page to introduce the Oneness Kingdom program.
- User Registration: Allow new users to register an account.
- User Login: Enable existing users to log in securely to access the platform.
- Supabase Integration: PostgreSQL for transactional data, Auth, and Realtime.
- Neo4j Graph: Social relationships and Trust graph queries.
- Realtime Features: Chat, notifications, AI avatar state via Supabase Realtime.
- Marketplace: Listings, interactions, evaluations, and points ledger.

## Style Guidelines:

- Primary color: A harmonious blend of light purple (#B19CD9) to represent spirituality and unity.
- Background color: Very light purple (#F1E9FA) providing a calm and ethereal backdrop.
- Accent color: A soft blue (#ADD8E6) to highlight interactive elements and represent peace.
- Body and headline font: 'Alegreya', a serif font to reflect the platform's intellectual feel.
- Use gentle, flowing icons that represent love, peace, and harmony. Icons should be visually accessible.
- A clean, spacious layout with intuitive navigation to guide users through the initial steps of joining the Oneness Kingdom.
- Subtle animations and transitions to create a smooth and engaging user experience. Examples: loading screens featuring mandala patterns
